A **Pomodoro Timer** and **Stopwatch** application inspired by the "Upside Down" atmosphere of *Stranger Things*. Built with Python and PyQt5.

## Features

* **Upside Down Aesthetic:** Custom fonts and color schemes inspired by the show.
* **AI-Generated Art:** Features a unique, custom-generated background representing the "Upside Down" world.
* **Stopwatch:** Millisecond-precise tracking.
* **Focus Timer:** Customizable countdown timer (Pomodoro style).

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ietkose/stranger-things-clock-widget.git
    cd stranger-things-clock-widget
    ```

2.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the App:**
    ```bash
    python main.py
    ```

## Tech Stack

* **Language:** Python 3.x
* **GUI:** PyQt5
* **Assets:** Custom AI-generated background, Custom Fonts.
